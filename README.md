# 🏏 IPL Analytics Dashboard (2008–2024)

An interactive Tableau dashboard built using IPL match and ball-by-ball data from 2008–2024.

---

## 📊 Project Overview

This project analyzes 17 IPL seasons using Tableau to provide insights into:

- Team Performance
- Player Performance
- Venue Analysis
- Toss Impact
- Batting Statistics
- Bowling Statistics
- Season Trends

---

## 🚀 Dashboard Features

- Total Matches
- Total Runs
- Total Wickets
- Total Balls
- Average Runs
- Top Run Scorers
- Top Wicket Takers
- Strike Rate Analysis
- Economy Rate Analysis
- Toss Winner vs Match Winner
- Venue-wise Performance
